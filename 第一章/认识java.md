# 认识Java

Java语言是由“高司令”在1995年提出的一种面向对象的高级程序设计语言，其具有跨平台、分布式、健壮安全、多线程等特性，使用Java能够编写桌面应用程序、Web应用程序、分布式系统和嵌入式系统应用程序 。
